# Memory Domain Reference

## When to use each domain

### decisions/
Strategic choices with lasting impact. Minimum viable entry = Decision + Rationale + Date.
Include alternatives considered — future-you will ask "why didn't we do X instead?"

### clients/
One file per client organization. Accumulates over time: contacts, preferences, engagement history,
political dynamics, what they respond to. More valuable the longer you work with them.

### research/
Synthesis notes from reading, interviews, workshops. Not raw notes — distilled insights.
Ask: "What's the most important thing I learned?" and write that.

### trends/
Confirmed patterns with multiple data points across sources or time. Horizon-tagged.
Distinct from signals: a trend has evidence behind it.

### signals/
Single observations that might indicate something coming. Weak, ambiguous, early.
The goal: capture before it becomes obvious. Attribute the source.

### projects/
Engagement-level notes: scope, status, key decisions, team, client, milestones.
Keep a brief "current state" section updated — the rest is archive.

### lessons/
Hard-won knowledge. Transfer format: "On projects with [characteristic], [do/avoid] X."
Always include the counter-example: when does this lesson NOT apply?

### hypotheses/
Strategic bets being actively tracked. State the falsification condition.
Update status as evidence accumulates: Tracking → Validated / Invalidated.

### opportunities/
Possibilities worth developing. Stage-gated: New → Evaluating → Pursuing → Passed.
Include "Why now" — opportunities are time-sensitive.

### scenarios/
Coherent future states, not predictions. Each needs: drivers, triggers, implications, response.
Scenarios are tools for strategy, not forecasts.

### team/
Working styles, expertise, availability, trust level, communication preferences.
Useful when staffing or collaborating. Keep respectful — these are colleagues.

### capabilities/
Methods, frameworks, tools, and skills Sintetica has or is developing.
Links to which projects deployed them and with what results.

### innovation/
Raw ideas, experiments, things worth exploring. Low bar for entry. Tag status:
Idea / Exploring / Tested / Adopted / Dropped.
